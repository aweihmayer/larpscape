# ⚙️ Editable configurations

The app has configurations that you can change through the interface which will change certain behaviors.   
Configurations can only be changed by administrators.

| CONFIG | DESCRIPTION |
| :---- | :---- |
| app.lang |  |
| email.provider | Determines the provider for email services. |
| email.resend.api\_key | The API key used to send emails with Resend email provider. |
| email.system\_address | The email shown as the send for system generated emails. |
| payment.provider | Determines the provider for payment services for purchasing products. |
| users.characters.max | Determines the maximum amount of characters a user can create. |
| users.signup.min\_age | Determines the minimum age a user must have to create an account. |
| users.signup.open | Determines if guests are allowed to openly signup and create an account rather than doing so through a manager. |
| users.signup.requires\_confirmation | Determines if new users are required to activate their accounts via email. If this is enabled, an email provider must be set up. |
